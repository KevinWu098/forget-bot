# Forget Bot

A ~~minimal~~ mostly-minimal serverless bot for ~~reminders~~ forgetfulness on
Discord. Built on Vercel Workflows, Vercel KV (Redis), and modern Discord.js
(modals, interactions).

Code architected with Cursor, ChatGPT, DeepWiki, and CodeWiki, at a cost of ~$51.73 across four days (Dec 19 - Dec 23 2025).
